Features so  far - 
0. Authentication (sign up , sign in)
1. Youtube look alike landing page with recommendations of latest videos. 
2. Feature of Video uploading with description, title and thumbnail.
3.Watching videos online.
4.Features like Like , Subscribe and comment.
5.YouTube like channel pages of individual channel which opens on clicking on the channel logo.
6. Explore/ Trending Page shows videos with highest views.
7. Searching feature
8. Customizing your own channel page
--------------------------------------------------------------------------------------------------
HOW TO EXCUTE---
------------------

Follow the link below--
https://clonedevlooperproject.web.app/


Or follow the steps below...

Note : Please install NodeJs before trying for below steps

1.Download the code in your local system and save it in a particular folder.
2.Open that folder in Visual Studio Code.
3.Open new terminal
4.Commands to be followed..
 a.  npm install
 b.  npm run start
 


----------------------------------------------------------------------------------------------------

Issues may be faced on local host-

There might be some errors while playing the video while exiting the video.
These errors only come in the development server and does not affect the hosted site.
